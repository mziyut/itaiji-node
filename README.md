# itaiji-node

Inspired by [camelmasa/itaiji](https://github.com/camelmasa/itaiji).

## Installation

```sh
npm install itaiji
```

## Usage

```ts
import { seijitai } from 'itaiji';

seijitai('亞'); // -> "亜"
```
